# react版动态简历

> 

[预览](http://songhao888.cn/jianli/)

## 使用方法

``` bash
git clone https://github.com/songhaoreact/reactjianli.git
cd reactjianli
npm install
npm start
```

## 打包发布
```
npm run build
发布html文件和build即可
```
